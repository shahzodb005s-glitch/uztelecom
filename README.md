📌 About The Project

Uztelecom — это лендинг страница, посвящённая изысканному десерту Uztelecom, который известен своей лёгкой текстурой, хрустящей корочкой и нежным кремом .

Проект создан для практики фронтенд-навыков и демонстрации современного UI/UX дизайна.

✨ Features
⚡ Быстрая загрузка
📱 Полностью адаптивный дизайн
🎨 Чистый и современный UI
🧩 Семантическая HTML структура
💡 Простая и понятная навигация
